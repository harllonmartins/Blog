Welcome to my minimal blog! This is a super-fast, privacy-first blogging platform built with simplicity in mind.

## What makes this blog special?

- **No trackers** - Your privacy is respected
- **No JavaScript** - Fast, secure, and accessible
- **Minimalist design** - Content is king
- **Easy to maintain** - Just write markdown files
- **Static pages** - Lightning-fast performance
- **Mobile-friendly** - Looks great on any device

## Getting started

To add new posts, simply create a markdown file in the `posts/` directory with frontmatter containing:

- `title` - The post title
- `date` - Publication date
- `author` - Author name
- `tags` - Array of tags
- `excerpt` - Short summary

Then run `npm run build` to generate your blog!

Enjoy writing! ✍️
